# 눌림목 스크리너 (Pullback Scanner)

한국 주식 시장(코스피·코스닥) 종목의 **눌림목 패턴**을 자동 분석하는 웹앱입니다.

## ✨ 기능

- 🔍 **단일 종목 분석**: 종목코드 입력 → 즉시 0~100점 점수화
- 📊 **관심종목 일괄 분석**: 최대 30개 동시 분석, 점수 순 정렬
- 📈 **차트 시각화**: 종가 + 5/20/60일 이동평균선
- 💰 **매매 가이드**: 손절선·1차/2차 목표가 자동 계산
- 📱 **모바일 최적화**: 핸드폰에서도 깔끔하게 작동

## 🎯 점수 산정 (총 100점)

- **추세 확인 (30점)**: 20일선 우상향, 이평선 정배열, 직전 상승폭
- **조정 깊이 (25점)**: 5~25% 조정 + 피보나치 23.6~61.8% 되돌림
- **이평선 지지 (20점)**: 5일/20일/60일선 중 어디서 지지받는지
- **거래량 패턴 (15점)**: 조정 시 거래량 감소 여부
- **반등 신호 (10점)**: 거래량 동반 양봉 출현

## 📁 파일 구조

```
pullback_web/
├── app.py              # Flask 백엔드 (분석 로직 + API)
├── templates/
│   └── index.html      # 프론트엔드 (모바일 최적화 UI)
├── requirements.txt    # 파이썬 의존성
├── Procfile            # Render/Heroku 실행 명령
├── render.yaml         # Render 배포 설정
├── runtime.txt         # 파이썬 버전
├── DEPLOY.md           # 🚀 배포 가이드 (필독!)
└── README.md           # 이 파일
```

## 🚀 배포하기

**`DEPLOY.md` 파일을 열어보세요.** 30분 안에 배포 가능한 단계별 가이드가 있습니다.

요약:
1. GitHub에 파일 올리기 (드래그 앤 드롭)
2. Render에 연결 (클릭 몇 번)
3. URL 받아서 핸드폰에서 사용

**파이썬 설치, 명령어 입력 전혀 필요 없음.** 무료.

## 🛠 로컬 테스트 (선택, 개발자용)

파이썬이 설치된 PC에서:
```bash
pip install -r requirements.txt
python app.py
```
브라우저에서 http://localhost:5000 접속.

## 🎨 기술 스택

- **백엔드**: Python 3.11, Flask, pykrx
- **프론트엔드**: Vanilla JS, Chart.js
- **데이터**: KRX (한국거래소) 공식 데이터
- **배포**: Render.com (무료 플랜)
